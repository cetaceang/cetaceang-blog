## 关于 dedirock

这家商家全名是 Dedi Rock LLC，由一批有多年主机行业经验的人创立。按他们自己的说法，创始团队在做了一段时间网络营销之后，觉得主机市场普遍缺乏"以客户为先"的服务理念，于是花了一年多时间搭建 dedirock，目标是做到技术扎实、服务到位。

架构上用的是 OpenNebula 云平台，官方强调其冗余性和稳定性，并提到该平台被美军、纳斯达克等机构采用。除 VPS 外，也提供独立服务器、虚拟主机和机柜托管，销售和技术支持宣称 24/7 在线。

dedirock 是 ColoCrossing 下游。这也基本决定了它的定位：价格便宜，但“灵车”的风险实实在在——数据丢失、机器失联、网络线路拉跨，都是这类机器的常见问题，不是危言耸听。性能也别抱太高期望，计算资源有限，跑高负载不是它的场子。  

不过话说回来，价格都这么便宜了，还要什么自行车。预期放对、备份做好、关键业务别放上面，作为一台玩具鸡，或者新手的第一台鸡练手一下linux命令，搭建服务，它就是一台很有性价比的备选。另外，dedirock的客服服务响应快，而且态度好，确实符合他们宣称的以客户为先。

### 机型介绍

如名字所言，这个款是去年2025年的黑五促销机型（是的，现在还在售卖黑五促销机型）。机器的cpu性能偏弱，我的这台cpu是Xeon E5-2660 v2，不过我也有看到有人的cpu是比较新的Intel(R) Xeon(R) Gold 6226R，性能会强不少（更有性价比的灵车了有没有）。机器本身的线路极其炸裂，能连上就算胜利。ip的解锁从脚本看还算正常，不过我也有看到有人说机器在油管有广告和没广告之间来回横跳（有广告说明送中了），还是以实际体验为主。

另外他们家还有一款 1C/2.5G/15G 的机型，年付只要 $6.45，内存甚至多了 0.5G。不过 15G 硬盘实在太小了，装个系统加上日常日志就捉襟见肘，还是这款 30G 的用起来更舒服点。这款机器还有LA的机房，不过论坛反馈似乎NY的稳定性好不少，我还是买了NY的机器。

**优势：**
- **价格极低**：6.75 刀/年，一顿麦当劳的钱换一台能跑服务的 VPS，性价比拉满
- **2G 内存**：在这个价位段给到 2GB 内存非常良心，跑一些中间件服务绰绰有余
- **2TB 流量 + 1Gbps 端口**：流量和带宽在这个价位属于大方的
- **客服响应快**：工单回复速度快，态度也好，这一点确实难得

**劣势：**
- **CPU 拉跨**：E5-2660 v2 是上古 CPU，单核跑分只有 583，别指望跑什么吃性能的任务
- **线路极差**：三网回程延迟 230-330ms，丢包也多，裸连体验很差
- **ColoCrossing 下游**：灵车属性拉满，数据安全性没有保障，重要数据千万别放上面
- **IP 可能送中**：有用户反馈 IP 会在干净和被污染之间横跳，稳定性存疑

## 实际使用场景与体验

### 我用它做什么

这台机器在我手里主要用来部署 **AI 中间网关的反向代理服务**。机器本身的线路确实拉跨，但套上 Cloudflare 的 CDN 之后，这个反代也不是不能用：

- 部署了一些 AI 服务的中间网关，用来做请求转发和负载均衡
- 2G 内存跑这类轻量代理服务完全够用，内存占用不高
- CPU 虽然弱了点，但反代本身不太吃 CPU，转发请求不成问题
- 套上 CF 之后用户端不会直连那条炸裂的线路，API 请求走边缘节点，延迟可以接受
- 毕竟传输的都是文本数据，不是视频流，对速度的要求没那么高，套个 CDN 就够用了

当然，如果你想用来部署openclaw，也不是不行，取决于你要不要运行浏览器，如果只使用api的话，2g内存完全够用。运行浏览器，内存其实应该也够用，但是cpu行不行我也不知道，因为我目前没有部署openclaw并且对它不是特别感兴趣。而且，都这么便宜，如果openclaw不小心给你rm -rf /*，你也可以不心疼，反正也没有其它重要数据。

这台机器也适合跑一些离线定时任务，比如用 crontab 定时从其他服务器拉取备份数据，再通过 rclone 上传到 OneDrive、Google Drive 这类云端网盘。这种任务对延迟不敏感，线路差完全不影响；而且 2TB 的月流量对于备份来说绰绰有余，2G 内存跑 rclone 也完全够用。

另外，它让我的探针多了一个美东机器。让mjj的探针看起来更有实力了。满足了虚荣心。

### 使用体验

**稳定性**：目前使用了几天，机器本身没有出现宕机的情况，服务一直在线，cpu也没出现过异常的高占用。不过作为 ColoCrossing 下游，建议还是自己进行备份。当然，如果你是用来玩，没有什么重要数据，不进行备份也无所谓。总的来说，稳定性还需要更长时间的观察。

**网络体验**：裸连的话确实烂，但套上 Cloudflare 之后基本能用。API 请求的延迟在可接受范围内，毕竟走的是 CF 的边缘节点。偶尔会有波动，但不影响正常使用。ssh可能会有点卡顿，不过开个魔法ssh应该会有效缓解，我因为有跳板机，所以ssh对我来说延迟也可以接受。

**资源占用**：2G 内存跑反代服务绰绰有余，日常占用不到一半。CPU 虽然弱，但反代场景下基本不会成为瓶颈。30G 硬盘空间也够用，反代服务本身不占什么磁盘。

**适合场景**：
- ✅ 轻量反向代理（套 CDN 后体验尚可）
- ✅ API 网关 / 中间件部署
- ✅ 离线定时任务（备份数据、rclone 同步等，不吃延迟）
- ✅ 探针、监控等轻量服务
- ✅ 新手练手 Linux（便宜，搞坏了也不心疼）

**不适合场景**：
- ❌ 需要直连低延迟的服务（线路太差）
- ❌ 对数据安全有要求的业务（灵车属性）
- ❌ 高并发 / 高性能计算（CPU 太弱）
- ❌ 大文件存储或数据库（硬盘小、4K IO 一般）

## 总结：值得买吗？

> 📌 **快速判断**
>
> ✅ **推荐购买**：想要一台极致便宜的玩具鸡，跑轻量服务、练手 Linux、或者套 CDN 做反代
>
> ❌ **不推荐**：在意线路质量、需要数据安全、跑高性能任务

### 核心优势

1. **极致便宜** - 6.75 刀/年，在整个 VPS 市场都算是底价了
2. **2G 内存** - 这个价位给到 2GB 内存，跑轻量服务绰绰有余
3. **客服不错** - 响应快、态度好，遇到问题不至于叫天天不应
4. **套 CDN 可用** - 虽然裸连烂，但套上 Cloudflare 之后作为反代节点还是能用的

### 适合这些人

- **想要极致便宜的练手机器**：6.75 刀/年，搞坏了也不心疼，适合新手折腾
- **轻量反代需求**：套 CDN 之后做 API 网关、反向代理等场景够用
- **不在意线路的用户**：如果你的服务本身就套 CDN，或者不需要从国内直连，线路差不是问题
- **备用机 / 玩具机**：挂个探针、跑个 bot、搭个轻量服务，不指望它干大事

### 不适合这些人

- **在意线路质量** - 三网回程全绕路，延迟 230ms+，裸连体验极差
- **在意数据安全** - ColoCrossing 下游，灵车风险实实在在
- **需要高性能** - E5-2660 v2 单核 583 分，跑不了什么重活
- **对 IP 纯净度有要求** - IP 可能送中，风险评分中等

### 我的建议

**6.75 刀一年，就别挑剔了。** 这个价格买到的就是一台能用的灵车，预期管理好，它就是一台性价比很高的玩具鸡。

我的实际使用体验是：拿它跑 AI 中间网关的反代，套上 Cloudflare CDN，虽然线路本身拉跨，但 API 请求走 CF 边缘节点之后延迟可以接受，日常使用没什么大问题。2G 内存跑这类轻量服务也绰绰有余。

**关键是做好心理建设和数据备份**：这是灵车，随时可能翻车。重要数据别放上面，配置文件做好备份，关键业务不要依赖它。只要预期放对了，这 6.75 刀花得还是很值的。
